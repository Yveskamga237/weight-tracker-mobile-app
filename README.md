# Weight Tracker Android App

## Overview

This project is an Android mobile application designed to help users track their weight, set fitness goals, and monitor progress over time. The app includes user authentication, data storage, and notification features.

## Features

* User login and account creation
* Track daily weight entries
* Set and update weight goals
* View progress over time
* Edit and delete weight records
* SMS notification when goal is reached
* User settings and preferences

## Technologies Used

* Java (Android Development)
* SQLite (Local Database)
* Android SDK

## Project Structure

* Activities: Handle UI and navigation
* Database Classes: Manage user and weight data (SQLite)
* Models: Represent application data
* Utility Classes: SMS notification system

## How It Works

Users can create an account and log in. Once authenticated, they can enter daily weight data, set goals, and monitor progress. The app stores data locally using SQLite and triggers SMS notifications when goals are achieved.

## Key Highlights

* Full mobile application built using Android
* Implements CRUD operations with SQLite
* Includes authentication and user management
* Integrates SMS notifications for user engagement

## Future Improvements

* Enhanced UI/UX design
